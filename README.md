# Full-Stack Student Management System
A web application to manage student records using a Python/Flask backend and MySQL database.

## Tech Stack
- **Frontend:** HTML5, CSS3 (Responsive UI)
- **Backend:** Python, Flask
- **Database:** MySQL
- **Tools:** VS Code, MySQL Workbench

## Features
- Full CRUD operations to add and view student data.
- Responsive design for mobile and desktop.
- Secure database integration via MySQL Connector.
